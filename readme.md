# Gerador de Senha

Esse é um projeto feito em Ionic e Vue.js que implementa um gerador de senha. Com essa aplicação, você poderá gerar senhas aleatórias de acordo com as suas preferências.

## Funcionalidades

- Você pode escolher o comprimento da senha desejada.
- Você pode selecionar se a senha deve incluir letras maiúsculas, minúsculas, números e caracteres especiais.
- A senha gerada será exibida em tempo real na tela.

## Pré-requisitos

Antes de iniciar a execução da aplicação, certifique-se de ter o Node.js e o Ionic CLI instalados em sua máquina.

## Como executar

1. Faça o clone deste repositório para o seu ambiente de desenvolvimento.

2. No terminal, navegue até o diretório raiz do projeto.

3. Execute o seguinte comando para instalar as dependências necessárias:

```bash
npm install
```

4. Após a instalação das dependências, execute o seguinte comando para iniciar o servidor de desenvolvimento:

```bash
ionic serve
```

5. A aplicação será aberta em seu navegador padrão na url `http://localhost:8100`. Caso não seja aberta automaticamente, acesse essa url manualmente.

## Tecnologias utilizadas

- Ionic
